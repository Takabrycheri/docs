# Benvenuto

Questa è la documentazione dei progetti Takabrycheri. Usa il menu laterale per navigare tra le sezioni: aggiungi nuovi file in `docs/` e la sidebar verrà aggiornata automaticamente.

## Come aggiungere una pagina

1. Crea un file Markdown in `docs/`, ad esempio `docs/nuovo-progetto.md`.
2. Opzionale: aggiungi frontmatter per ordinamento (`sidebar_position`) o per impostare uno slug personalizzato.
3. Fai commit e push su `main`: il sito verrà rigenerato e pubblicato su GitHub Pages.
